# PokemonTeamAnalyzer


Analyseur d'équipe pokémon.
Rentrez votre équipe et regardez contre quels types elle sera efficace ou non.

## Super efficace = +1
## Extrêmement efficace = +2
## peu efficace = -1
## inneficace = -2
